# 🏫 Si Aset (Sistem Informasi Aset Sekolah)

[![Laravel](https://img.shields.io/badge/Laravel-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)](https://laravel.com)
[![MySQL](https://img.shields.io/badge/MySQL-4479A1?style=for-the-badge&logo=mysql&logoColor=white)](https://www.mysql.com/)
[![Railway](https://img.shields.io/badge/Railway-0B0D0E?style=for-the-badge&logo=railway&logoColor=white)](https://railway.app/)

**Si Aset** adalah aplikasi sistem informasi berbasis web yang dirancang untuk mengelola inventaris, pengadaan, dan pendataan barang milik sekolah secara transparan dan akuntabel.

---

## 🚀 Fitur Utama

-   **Manajemen Inventaris (Admin/Petugas):**
    -   Input data barang lengkap (Merk, Kode Barang/SN, Tahun Perolehan).
    -   Kategorisasi barang (Elektronik, Meubel, Alat Peraga, dsb).
    -   Monitoring status kondisi barang (Baik, Rusak Ringan, Rusak Berat).
    -   Manajemen lokasi aset per ruangan (Kelas, Lab, Perpustakaan).
-   **Modul Pengadaan:** Pencatatan riwayat pembelian barang baru dan upload bukti nota/dokumen.
-   **Dashboard Publik:** Halaman transparan untuk masyarakat melihat statistik total aset dan kondisi barang sekolah.

---

## 🛠️ Stack Teknologi

-   **Framework:** Laravel (PHP)
-   **Database:** MySQL
-   **Server:** FrankenPHP / Nginx
-   **Deployment:** Railway
